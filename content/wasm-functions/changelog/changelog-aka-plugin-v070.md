### ✨ What's New in `spin aka` v0.7.0

This release brings the following improvements:

- **Secure credential storage**: login credentials are now safely stored in the operating system's native secret store (macOS Keychain, Linux secret service, or Windows Credential Manager), keeping them off disk and out of plain-text config files. Existing credentials are migrated automatically on first use.
- **Rebranded to Akamai Functions**: CLI help text and messaging now reflects the Akamai Functions name.
<!-- break -->

Existing users can get started by upgrading the [`spin aka` plugin](https://spinframework.dev/v3/managing-plugins#upgrading-plugins) or [request access](https://fibsu0jcu2g.typeform.com/fwf-preview) to Akamai Functions today.
